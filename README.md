## 網頁介紹

### 基本架構
* 前台:
    * 首頁-商店介紹
    * 產品頁-多項商品分類以及單一商品細節、推薦其他商品、放入購物車
    * 購物車-使用酷碰碼、調整商品/查看購物車商品項目，無產品時推薦其他商品
    * 下單 / 結帳
    * 登入頁-登入後台介面
* 後台:
    * 產品頁-新增/刪除/修改產品
    * 訂單頁-修改/查看訂單
    * 酷碰頁-新增/刪除/修改酷碰
